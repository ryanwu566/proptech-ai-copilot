# Product Roadmap

本文件用來切分後續產品開發優先順序，不代表下列功能已完成。TaxOracle 規則引擎維持既有穩定主線；估價、地圖與風險模組皆為決策輔助，不宣稱精準預測、正式鑑價、核貸或法律結論。

## Phase 1：估價可信度與資料查詢

- 六都與其他縣市 rolling 3 年資料覆蓋狀態
- 可比成交篩選與來源透明化
- 市場趨勢與保守／中性／樂觀情境
- 資料品質與異常期間提示
- 估價頁手機響應式與寬表格可讀性

完成標準：使用者能清楚知道本次使用哪些官方資料、估價信心限制，以及資料是否足以支持路段或行政區層級判讀。

## Phase 2：使用者功能

- 收藏估價案例
- 匯出估價摘要 PDF 或 HTML
- 估價條件分享連結
- 最近查詢紀錄
- 社區／建案名稱搜尋優化

以上功能仍需先完成資料安全、權限與隱私設計；目前只列入 roadmap，不在本階段實作。

## Phase 3：進階功能

- 地圖估價
- 生活機能／POI
- 行政區趨勢比較
- 投資型指標
- 租金收益率
- 貸款月付試算
- 稅費與持有成本整合

所有進階指標都應清楚揭露資料來源、計算方式與限制，不應包裝為成交保證、投資建議或正式金融／稅務結論。
